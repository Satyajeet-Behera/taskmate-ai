# 🤖 TaskMate AI

TaskMate AI is an intelligent productivity platform designed to help users organize tasks, manage schedules, and improve efficiency using AI-powered assistance.

## 🚀 Live Demo

https://satyajeet-taskmate-ai.netlify.app

## ✨ Features

- AI-powered task assistance
- Smart scheduling
- Task prioritization
- Responsive design
- Modern dashboard
- Productivity analytics

## 🛠️ Tech Stack

- React
- Tailwind CSS
- Supabase
- Gemini AI
- JavaScript
